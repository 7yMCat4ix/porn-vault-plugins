## adultempire 0.1.0

by boi123212321

Scrape data from adultempire

### Arguments

| Name | Type    | Required | Description                    |
| ---- | ------- | -------- | ------------------------------ |
| dry  | Boolean | false    | Whether to commit data changes |

### Example installation

```json
---
{
  "PLUGINS": {
    "adultempire": {
      "path": "./plugins/adultempire/main.js",
      "args": {
        "dry": false
      }
    }
  }
}
```

```yaml
---
PLUGINS:
  adultempire:
    path: ./plugins/adultempire/main.js
    args:
      dry: false

```
